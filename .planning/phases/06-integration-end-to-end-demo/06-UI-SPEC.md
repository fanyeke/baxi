# Phase 6 — UI Design Contract

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

**Phase type: Integration & Bugfix (no new UI components)**

This phase has zero new UI features. All changes are:
1. **DTO type alignment** — Frontend TypeScript types changed to match actual backend JSON response shapes
2. **Test assertion fixes** — Failing tests updated to assert text that components actually render
3. **Pipeline request body fix** — Field name `pipeline_type` changed to `config` in mutation payload

No new components, pages, dialogs, or visual patterns are introduced. This contract documents the existing design system baseline and the specific copy/type changes required.
