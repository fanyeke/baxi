       a. Extract case_id from chi.URLParam(r, "case_id")
       b. Call h.svc.Compare(r.Context(), caseID)
       c. Handle errors: pgx.ErrNoRows → 404, other errors → 500
       d. On success, transform *eval.DecisionComparison into dto.CompareResponse:
          - DecisionCaseID = comparison.DecisionCaseID
          - Metadata.DecisionTypeMatch = comparison.DecisionTypeMatch
          - Metadata.SeverityMatch = comparison.SeverityMatch
          - Metadata.ActionOverlap = comparison.ActionOverlap
          - Metadata.ConfidenceDiff = comparison.ConfidenceDiff
          - Metadata.CreatedAt = comparison.CreatedAt.Format(time.RFC3339)
          - Build Added array: if !comparison.DecisionTypeMatch, add DiffItem{Field: "decision_type", Before: comparison.RuleDecisionType, After: comparison.LLMDecisionType, ChangeType: "changed"}
          - Build Changed array: if !comparison.SeverityMatch, add DiffItem{Field: "severity", ChangeType: "changed"}
          - If comparison.ConfidenceDiff > 0, add DiffItem{Field: "confidence", Before: nil, After: comparison.ConfidenceDiff, ChangeType: "changed"}
          - If comparison.ActionOverlap < 1.0, add DiffItem{Field: "actions", Before: comparison.ActionOverlap, After: nil, ChangeType: "changed"}
          - Set Added and Removed to empty arrays (not nil) for consistent response shape
       e. Return with httputil.JSON(w, http.StatusOK, resp)

    2. Implement Replay handler (replaces the 501 stub at line 276-278):
       a. Extract case_id from chi.URLParam(r, "case_id")
       b. Parse optional request body into dto.ReplayRequest:
          - Use json.NewDecoder(r.Body).Decode(&req)
          - If decode fails, default to dry_run=false (do not return 400)
       c. Call h.svc.Replay(r.Context(), caseID, req.DryRun)
       d. Handle errors: if error message contains "not configured" or "replay service not configured" → return 503 Service Unavailable with message "replay service not available"; pgx.ErrNoRows → 404; other errors → 500
       e. On success, transform *eval.ReplayResult into dto.ReplayResponse:
          - OriginalDecision = structToMap(result.OriginalDecision)
          - ReplayedDecision = structToMap(result.ReplayedDecision) if not nil
          - ContextHash = result.ContextHash
          - Model = result.Model
          - DryRun = result.DryRun
          - If result.Diff != nil, build ReplayDiff from eval.DecisionDiff fields
       f. Return with httputil.JSON(w, http.StatusOK, resp)
       g. Note: The service's Replay method does not support model/temperature/context_overrides yet (per deferred items in CONTEXT.md). For now, parse these fields from the request but only pass dryRun to the service. Document this in a code comment: "Model/temperature/context_overrides are parsed for future use; currently only dry_run is passed to the replay service."
