    ## 迁移模式概述

    **模式 A（shim 作为构造函数参数）**：对于那些接受 `*repository.XxxRepository` 的服务和领域对象，更新为接受子包类型（如 `*governanceRepo.Repository`），并移除所有方法调用中的 `pool` 参数（因为 PoolProvider 已嵌入）。

    **模式 B（interface 参数）**：对于那些通过 interface 调用的，更新 interface 方法签名以移除 `pool` 参数，并将返回值中的 `repository.TypeName` 改为子包类型。

    **模式 C（类型别名）**：将 `repository.XxxRow`、`repository.XxxFilters` 等类型引用改为子包类型。
