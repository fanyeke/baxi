| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-07 | Information Disclosure | handler/{decision,action,outbox,pipeline,review,sandbox}.go | mitigate | writeServiceError 使用增强的 classifyError 隐藏 DB 内部细节；writeServiceError 使用通用消息而非原始错误消息 |
| T-02-08 | Tampering | handler/*.go — writeValidationError | mitigate | 验证错误仅包含指定的字段名和预定消息，不反射/不暴露内部状态 |
